# Privacy Policy — Safety Pyramid Power BI Visual

**Last updated:** March 16, 2026

## Data Collection

The Safety Pyramid visual **does not collect, store, transmit, or share any user data**. All data processing occurs entirely within the user's browser or Power BI environment.

## External Services

This visual **does not make any network requests** to external services, APIs, or endpoints. It operates completely offline once loaded.

## Data Processing

The visual receives data exclusively through the Power BI DataView API. This data is used solely for rendering the pyramid visualization on screen. No data is cached, persisted, or sent outside the Power BI host environment.

## Cookies and Local Storage

This visual **does not use cookies, local storage, or any other browser storage mechanism**.

## Third-Party Libraries

The visual uses the following open-source library for rendering:
- **D3.js v7** (BSD 3-Clause License) — used exclusively for SVG rendering within the visual container.

No third-party analytics, tracking, or advertising libraries are included.

## Contact

For questions about this privacy policy, please open an issue at:

## Changes to This Policy

Any changes to this privacy policy will be reflected in this document with an updated date.
